# ReliefLink

ReliefLink is a public service platform built with TypeScript and Firebase that connects communities during emergencies by facilitating disaster relief coordination and blood donation support.

The system enables users to request help, volunteer, or register as donors, while also using AI-driven suggestions to guide users toward the most suitable role based on their profile and activity.

---

## Core Features

* Secure user authentication (Email/Password and Google Sign-In)
* User profile creation and management
* Emergency request submission (medical aid, rescue, disaster relief, blood requests)
* Volunteer registration system
* Blood donor registration and management
* AI-based role suggestion system for users
* Real-time data handling with Firebase

---

## AI-Powered Feature

* Smart agent analyzes user profile and activity
* Suggests appropriate roles such as volunteer, donor, or requester
* Improves community engagement during emergencies

---

## Backend & Database

* Firebase Authentication for secure login and identity management
* Firebase Firestore for storing user data, requests, and registrations
* Real-time updates for emergency coordination
* Scalable cloud-based architecture

---

## Tech Stack

* TypeScript
* Firebase Authentication
* Firebase Firestore
* Firebase Hosting (if applicable)
* Modern web development practices

---

## Project Goal

The goal of ReliefLink is to create a fast, reliable, and accessible platform that connects people in need with those who can help, ensuring efficient coordination during critical situations such as disasters and medical emergencies.
